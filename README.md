
# Personal Expense Analysis

## Overview
This project analyzes personal expenses using Python. It helps users track spending patterns, calculate total expenses, and visualize expenses by category.

## Features
- Load expense data from a CSV file
- Calculate total expenses
- Analyze category-wise spending
- Generate charts for expense visualization

## Technologies Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset
The dataset contains:
- Date
- Category
- Amount

## How to Run
1. Upload the CSV file.
2. Run the Python notebook/script.
3. View the expense summary and charts.

## Output
- Total Expenses
- Category-wise Expense Analysis
- Expense Distribution Chart
